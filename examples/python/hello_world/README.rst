About
=====

The client sends three greetings to the server.

.. code-block:: text

   $ make
   ...
   $ ./server
   ...
   Received 'Hello!'.
   Received 'Hi!'.
   Received 'Hi again!'.

Run the client in another terminal.

.. code-block:: text

   $ ./client
   Connected to the server.
   Sending 'Hello!'.
   Sending 'Hi!'.
   Sending 'Hi again!'.
   Disconnecting from the server.
   $
